Introduction to Database

Level 1 CST7215_300

Databases are used to store data and are a core component of many information technology systems. I learnt the fundamentals of relational databases design using Entity Relation Diagrams (ERDs), and use Structured Query Language (SQL) to create, modify and query a database. I designed and created databases that are maintainable, secure and adaptable to change in business requirements, using normalization.
